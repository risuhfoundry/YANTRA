# Machine Learning Core Path

Machine Learning Core is the path where the learner moves from general AI intuition into actual prediction systems that learn patterns from data.

This part of the roadmap should cover the central ideas behind classical machine learning:

- supervised learning
- training and testing
- features and labels
- loss and optimization
- generalization
- common failure modes like underfitting and overfitting

The main goal is not memorizing model names. It is understanding the workflow:

1. define a task
2. collect or prepare data
3. represent the inputs as features
4. train a model
5. evaluate performance
6. improve the system based on errors

Useful examples for this path:

- email spam vs not spam
- house price prediction
- sentiment classification
- recommendation or ranking basics

Yantra should teach machine learning as a process of improving predictions through evidence. It should avoid giving the impression that a model is "smart" in a human sense. The learner should leave with a clear grasp of why data, features, evaluation, and iteration matter more than hype.

This path should prepare the learner for deeper topics like gradient descent, neural networks, embeddings, and transformers.
