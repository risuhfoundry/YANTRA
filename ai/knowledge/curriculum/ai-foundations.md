# AI Foundations Path

AI Foundations is the starting path for learners who need the mental model behind modern AI before jumping into advanced machine learning or model-building.

This path should help a learner answer questions like:

- what AI actually is and is not
- the difference between rules, data, and learned behavior
- how models make predictions
- why data quality matters
- what training, evaluation, and inference mean

The purpose of this path is not to overwhelm the learner with deep math on day one. It is to build stable intuition so later topics like embeddings, transformers, and backpropagation make sense instead of feeling magical.

Core outcomes for this path:

- explain AI in plain language
- describe a dataset, label, feature, and prediction
- understand why examples matter in training
- recognise common AI tasks such as classification and generation
- connect ideas to the rest of the Yantra roadmap

Good beginner lesson flow:

1. what AI is
2. python basics for data and models
3. simple machine learning ideas
4. neural network intuition
5. ethics, limits, and evaluation

When Yantra teaches from this path, it should use concrete examples such as spam filtering, image labels, or movie recommendations. It should avoid acting as if the learner already knows advanced statistics or matrix calculus.

The AI Foundations path should feel like the bridge from curiosity to confident first-principles thinking.
