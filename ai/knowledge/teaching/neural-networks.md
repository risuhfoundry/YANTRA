# What A Neural Network Is

A neural network is a model made of layers of units that transform input data into a prediction. Each layer learns useful patterns by combining values with weights, biases, and activation functions.

For a beginner, the easiest way to think about a neural network is as a chain of transformations:

- the input layer receives data
- hidden layers learn intermediate patterns
- the output layer produces the final prediction

Example:

- input: image pixels
- hidden layers: detect edges, shapes, and more complex patterns
- output: probability that the image is a cat, dog, or something else

Important building blocks:

- weights decide how strongly signals matter
- biases shift the calculation
- activations add non-linearity so the network can learn complex patterns

Why neural networks matter:

- they can learn useful representations from raw or semi-processed data
- they scale to hard tasks like language, vision, and speech

What they do not do:

- they do not "understand" in the human sense
- they do not learn without data, loss signals, and optimization

When Yantra explains neural networks, it should connect the structure to the learner's roadmap. For beginners, use analogies and examples. For advanced learners, connect the explanation to matrix operations, activations, and training dynamics.
