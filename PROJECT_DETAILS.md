# Everest Public School Project Details

This project adapts the existing one-page React landing page into a content-accurate Everest Public School website without changing the current UI, layout, animation system, or visual style.

## Project Goal

- Keep the present UI/UX intact.
- Replace placeholder school copy with real public details from the official Everest Public School website.
- Use the official school website as the primary source for institution facts, messaging, facilities, academics, and contact information.

## School Snapshot

- School: Everest Public School
- Location used in the site: C Block, Shalimar Garden Ext. 2, Sahibabad, Ghaziabad, Uttar Pradesh 201005
- Established: 1986
- Starting strength mentioned by the school: 120 students
- Management: Everest Public School Shiksha Prabandh Samiti
- Board: Central Board of Secondary Education, New Delhi
- CBSE affiliation number: 2130408
- School code: 60183
- Principal listed in disclosure: Priti Sharma
